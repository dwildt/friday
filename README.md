# Should I Deploy on Friday?

A simple, fun website to help you decide if you should deploy on a Friday.

## About

This is a vanilla HTML/CSS/JS website with an 8-bit blue theme. It provides funny answers to the age-old question: "Should I deploy on Friday?". The site supports English, Portuguese (Brazil), and Spanish.

This project was built with Gemini.

## Setup

1.  Clone the repository.
2.  Install the dependencies:
    ```bash
    npm install
    ```

## Available Scripts

### `npm start`

Runs the app in development mode by starting a local server. Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run lint`

Runs the linter on all JavaScript files in the `js/` directory.
